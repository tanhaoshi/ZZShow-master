### 简介
**吱吱**：一款集合了新闻、视频和图片信息的Material Design风格的开源应用。吱吱采用MVP模式，基于RxJava+Retrofit2+ijkplayer等优秀的第三方开源库开发。

**说明**：新闻模块数据和视频模块数据来源于网易，图片模块数据来源于干货集中营，在此表示感谢！若内容有侵权，请联系作者进行删除处理。 本程序仅供测试学习，用于其他用途所造成的纠纷与本人无关。
